4.0 - Global Mixins

4.1.0 - GLOBAL MIXINS

The global mixins

FUNCTIONS

  • font-size($font-size: $FontSize-b):
    Outputs a rem font size witha px value for a fallback

  • conditional($variable, $state: true):
    Allows quick conditional code setup. If $variable evaluates
    to true the content is shown. If $state is set to false then content is shown when $variable is false

  • respond-to($respondMinWidth, $respondOrientation: false):
    Media query mixin which can be used to respond to min widths and orientations

  • limit-to($limitMinWidth, $limitMaxWidth, $limitOrientation: false):
    Media query mixin which can be used to limit to min widths and max widths and also orientations
